Online Shopping Cart System (Java)
Project Description
The Inventory Management System is a console-based Java application designed to manage and organize different categories of products efficiently. The system supports multiple product types such as Electronics, Clothing, and Food, each with their own specific attributes.

This project demonstrates key programming concepts, including:

Object-Oriented Programming (OOP)
Inheritance and method overriding
Encapsulation and data abstraction
File Handling
Exception Handling
Installation Instructions
Prerequisites
Java Development Kit (JDK 8 or higher)
A code editor or IDE (e.g., VS Code, IntelliJ IDEA, Eclipse)
Setup Steps
Clone the repository: git clone Or download the project as a ZIP file and extract it.

Navigate to the project directory: cd Inventory-Management-System

Compile the Java files: javac *.java

Run the application:

java Main
Usage Instructions
After running the program, the application initializes an inventory with sample product data. The system allows you to:

Add and manage products
Handle different product categories
View product details such as price, quantity, and category-specific attributes
Generate a receipt.txt during checkout
